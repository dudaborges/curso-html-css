Noopener e Noferrer

Clique no link para acessar o canva

rel="noopener" é essencial quando for utilizar o target="_blank", por questão de segurança. Impede que um invasor mal intencionado, do link vinculado, usa o window.openerobjeto para alterar o conteúdo

rel="noferrer" tem um propósito semelhante e adicionalmente impede que o navegador envie o endereço da página para o navegador destinado. É ideal que se utilize os dois
